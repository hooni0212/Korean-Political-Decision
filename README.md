# Korean Political Decision

한국 정치인 카드게임의 규칙과 기획 문서를 관리하는 저장소입니다.

## 문서

- [한국 정치인 카드게임 규칙 v0.4](docs/rules-v0.4.md)
- [한국 정치인 카드게임 개발 명세서 v0.4](docs/development-spec-v0.4.md)

## 개발 기준

- Steam PC 출시를 1차 목표로 하며, 클라이언트와 룰 엔진은 Unity/C# 기준으로 문서화합니다.
- 모든 작업은 시작 전 관련 문서를 확인하고, 완료 후 구현과 문서의 차이를 정리하는 것을 기본 지침으로 합니다.
- PM 1명, 개발자 2명, QA 2명의 멀티 에이전트 협업 방식은 개발 명세서에 정의합니다.
